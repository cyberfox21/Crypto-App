# CryptoApp

Simple app about cryptocurrency used https://coinpaprika.com/ api

- Clean architecture
- Adapter Delegate pattern with DiffUtil
- Single Activity
- MVVM
- Flexbox layout
- Dagger Hilt
- Retrofit
- RxJava
